# GasSplitter

## Divide and conquer the roads. Carpool style.

By: Jibran Absarulislam, Colbe Roberson, & Mostafa Damir

### Installation steps:

```
git clone https://github.com/It-s-Saturday/GasSplitter.git
cd GasSplitter
```

virtualenv

```
python -m venv venv
pip install -r api/requirements.txt
```

Flask Back-end

```
cd flask
python app.py
```

React front-end

```
# in a new terminal

cd client
npm install
npm run start
```

### Technologies

Stack
- React
- Flask
- CockroachDB

APIs
- Google Cloud: googlemaps
- MPG API
- Cars API (API Ninjas)
- Twilio
